<ModAsset location="oritech:chainsaw" /> Piła łańcuchowa to narzędzie, które można naładować i jest niezniszczalne, pełniąc funkcję siekiery.  
Można ją również zaklinać. Aby naładować piłę łańcuchową, umieść ją w bloku [magazynowania energii](../logistics/energy) lub załóż [pancerz exo](../tools/exo_armor).
