Węzły zasobów można znaleźć na całym świecie, występują w skupiskach tuż nad warstwą bedrocka.  
Zawierają wszystkie typy rud z overworlda i zazwyczaj są przykryte małą kieszenią [surowej ropy](../resources/crude_oil).  

Węzły zasobów można wydobywać tylko za pomocą [bedrock extractor](../interaction/deepdrill).  

Na powierzchni lokalizację skupiska węzłów zasobów można rozpoznać po występowaniu [ore boulder](../resources/ore_boulder).  
Ore boulder wskazuje również rodzaj rud występujących w tym skupisku.
